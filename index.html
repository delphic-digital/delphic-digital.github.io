Tools

First things first, get the tools! Most 3rd party front end tools are based off of Node.js. So:

  • Install Node.js!

Once node is installed, we can install Volo. It's a great way to manage JavaScript libraries and dependencies. It's almost perfect for our uses, and it's perfect after a couple changes.

npm install -g volo

It doesn't allow you to run the create command in a directory that has code in it. This becomes a problem when us FED's get a new site to work on that already has a C# project set up. So lets make some small changes to Volo to fix this:

Comment out the else if starting on line 56 in C:\Users\[your username]\AppData\Roaming\npm\node_modules\volo\commands\create.js.

/*else if (file.exists(appName)) {
            return new Error(appName + ' already exists.');
        }*/

The other problem is that volo is built for JavaScript, so it can't, say...automate where to put a css dependency. Let's change this:

Comment out line 164 in C:\Users\jmills\AppData\Roaming\npm\node_modules\volo\lib\packageJson.js and add return path.dirname(this.file).

//return baseDir;
        return path.dirname(this.file);

Modifications done!
